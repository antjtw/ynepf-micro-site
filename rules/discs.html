Discs (weights)

This is a plain-English guide. The official IPF Technical Rules are the final word.

Discs are the round weights loaded onto the bar. Each one must be accurate, clearly marked, and follow the colour code.

Colour code

The main discs follow a colour code so referees can read the weight from the platform.

Disc colours by weight
WeightColour
25 kgRed
20 kgBlue
15 kgYellow
10 kg and underAny colour

How discs are loaded

  • The heaviest discs go on first, nearest the middle, then lighter ones outward.
  • The first disc on each side faces in, the rest face out.
  • The weight on each disc must be readable from the platform.
  • Each disc must be accurate to within 0.25% or 10 grams.

Competition discs run from 0.25 kg up to 25 kg. The biggest disc is no more than 45 cm across.

Record attempts

For a record, lighter discs can be used to make a weight at least 0.5 kg over the current record.
